# Introduction

Welcome to your book! This is a simple introduction chapter.

## What is this?

go-pretty-pdf transforms your MDX files into a beautiful, print-ready PDF.
Just write your content in `.mdx` files, organize them with `[chapter.section.subsection]` IDs,
and run a single command.

## How it works

1. Write your content in `.mdx` files inside this directory
2. Customize `go-pretty-pdf.yml` with your book settings
3. Add variables in `vars:` for substitution with `{{variable}}` syntax
4. Run `pretty-pdf check` to validate your content
5. Run `pretty-pdf build` to generate the PDF

<DeepDive title="Pro Tip">
Use `pretty-pdf check` frequently as you write to catch issues early.
It validates frontmatter, IDs, heading depth, and more.
</DeepDive>

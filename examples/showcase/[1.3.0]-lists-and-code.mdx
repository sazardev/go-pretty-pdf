# Lists & Code

## Unordered

- Parse MDX with frontmatter
- Transpile custom components
- Compose HTML with a template + CSS
- Render to PDF via headless Chrome

## Ordered

1. Write your `.mdx` files
2. Organize them with `[X.Y.Z]` IDs
3. Run `pretty-pdf build`
4. Get your PDF

## Nested

- Core pipeline
  - Parser (goldmark)
  - Composer (html/template)
  - Renderer (chromedp)
- CLI
  - `build`, `check`, `init`, `watch`

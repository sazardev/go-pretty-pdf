# Custom Components

Every component below is plain Go: a function that takes attributes and
inner content and returns HTML. None of this is built into the library —
it's registered once via `WithComponent` before `Build()` runs.

<DeepDive title="One attribute, many uses">
The transpiler currently captures a single `title` attribute per
component. Every component here reuses it for whatever single parameter
it needs: a level, a label, a date, or a percentage.
</DeepDive>

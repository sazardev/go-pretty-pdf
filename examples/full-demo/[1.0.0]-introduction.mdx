# Introduction

Welcome to **{{product}} v{{version}}** by {{company}}.

{{product}} transforms your `.mdx` files into a beautiful, print-ready PDF with a single command.
This example demonstrates **every feature** the tool offers.

<DeepDive title="What is {{product}}?">
It is a Go library and CLI tool that parses MDX files,
transpiles custom components, composes HTML with a template + CSS,
and renders a PDF via headless Chrome.
</DeepDive>

## Philosophy

Write your content in `.mdx` files. Organize them with `[X.Y.Z]` IDs.
Run `pretty-pdf build`. Get your PDF. That's it.
